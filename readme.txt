Restrict media library access based on user roles.

== Description ==

Media Restriction allows administrators to restrict media library access based on user roles. Users with restricted roles will only be able to see their own media files.

Features:
* Restrict media access by user role
* Exclude specific users from restrictions
* Compatible with both grid and list views of the media library

== Installation ==

1. Upload the `media-restriction` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Media Restriction' menu to configure settings

== Frequently Asked Questions ==

= Will this affect administrators? =

No, administrators and users with the capability to edit others' posts are not affected by the restrictions.
